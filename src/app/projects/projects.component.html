GE Additives Material Project

GE Digital (September - December 2017)

During my second internship with GE Digital I worked on a application for GE Additives. The project was to create a web application to store all GE's material catalog data and provide a place for GE customers and testers to be able to view the material catalog.

For this project I worked on the front end of the website. The framework we used for the front end were Angular2. During the project I learned more about using typescript, as I worked on a tool for parsing excel sheets with data into the Material database. I also worked on the overall styling of the web application.

DTE PowerClerk

DTE Energy (January - May 2017)

At my engineering Co-op at DTE Energy I worked on a new application system for customers to acquire and build renewable energy sources into their businesses or homes. I specifically responsible for migrating the old data into a new build system called PowerClerk, that would allow for more automation for the customer and engineers involved in the project.

During this project, I was mainly responsible for migrating our data from the old system into the new one. Most of the data was collected through CSV files, that I would obtain from SQL queries from the old database. After this, much of the data was missing or incorrect values for some projects, so I would have to parse through the data using excel.
